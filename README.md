# Simple Interest Calculator

A simple command-line tool or script that calculates the interest earned over a period of time.

## Formula used
The calculator uses the standard simple interest formula:
$A = P(1 + rt)$

Where:
* **A** = Final Amount
* **P** = Principal amount (initial investment)
* **r** = Annual interest rate (decimal)
* **t** = Time (years)

## How to use
1. Input the Principal amount.
2. Input the annual interest rate.
3. Input the time in years.
4. The calculator will output the total interest and final balance.
